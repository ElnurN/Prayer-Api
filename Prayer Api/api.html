Fatihə surəsi:
  • Bismillâhir Rahmənir Rahim
  • Əlhəmdu lillahi Rabbil-âlemin
  • Ər-rehmənir rehim
  • Məliki youmiddin
  • İyyəkə nə'budu və İyyəkə nəstəin
  • İhdinas siratal mustəqim
  • Siratal ləzinə ən'əmtə aleyhim
  • Ğayrli məğzubi aleyhim vələzzallin
İxlas surəsi:
  • Bismillahir-rəhmanir-rəhim
  • Qul huvəllahu əhəd
  • Allahus-saməd
  • Ləm yəlid və ləm yuləd
  • Və ləm yəkun ləhu kufuvən əhəd
Təşəhhüd:

Əlhəmdu lillah. Əşhədu ənla ilahə illəllah, vəhdəhu la şərikə ləh. Və əşhədu ənnə Muhəmmədən abduhu rəsuluh. Allahummə salli əla Muhəmmədin və ali Muhəmməd.

Salamlar:

Əssəlamu əleykə əyyuhən-nəbiyyu və rəhmətullahi və bərəkatuh. Əssəlamu əleyna və əla ibadillahis-salihin. Əssəlamu əleykum və rəhmətullahi və bərəkatuh.

Diqqət
  • 1. Bu əməllərin cəminə rükət deyilir. Fatihə surəsi + İxlas surəsi + rüku + 2 dəfə.
  • 2. Təşbihati ərbəə 3-cü və 4-cü rükətdə Fatihə surəsinin əvəzinə deyilir.
  • 3. Sübh namazının qılınma vaxtı dan yeri söküləndən günəş çıxana qədərdir.
  • 4. Zöhr və Əsr namazlarının qılınma vaxtı günortadan gün batana qədərdir.
  • 5. Şam və Xiftən namazlarının qılınma vaxtı gün batandan gecə yarısına qədərdir.
  • 6. Səcdə zamanı alının möhür (və ya torpaq, daş və bu kimi şeylərin) üzərində olması vacibdir.
  • 1-ci gün: Ya Qadyəl hacət
  • 2-ci gün: Ya Ərhəmər-rahimin
  • 3-cü gün: Ya Həyyu ya Qəyyum
  • 4-cü gün: La ilahə illəllahul məlikul haqqul mubin
  • 5-ci gün: Allahummə səlli alə Muhəmmədin və ali Muhəmməd əccil fərəcəhum
  • 6-cı gün: Ya Rəbbəl aləmin
  • 7-ci gün: Ya zəl cəlali vəl ikram
